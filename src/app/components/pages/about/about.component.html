Rick and Morty GraphQL APP

Angular 12 + Rick and Morty GraphQL API.

Proyecto realizado siguiendo el curso de Domini Code.
Para utilizar la API de Rick and Morty GraphQL, se instaló Apollo Angular para poder interactuar con las consultas en GraphQL mediante querys y poder obtener los datos de la API.

Así mismo para el Scroll Infinito se utilizó ngx-infinite-scroll y ngx-Toastr para las notificaciones.
