## Implemented Dependencies: 
* Apollo-angular
```
ng add apollo-angular
```
* nguniversal/express-engine
```
ng add @nguniversal/express-engine
```
* Ngx-Infinite-Scroll
```
  npm i ngx-infinite-scroll@13.0.1
```
* Ngx-Toastr
```
npm i ngx-toastr@14.1.1
```
